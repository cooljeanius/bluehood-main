Scope Component

  1. Description

    2. It observe a signal. It is like to oscilloscope.

  2. Ports

    1. In Ports

      • in_: The observed object.

    2. Out Ports

      • thru: The signal passed through.

  3. User Interface

    4. It runs as the following 3 modes.

    • Free Run: It always observes.
    • Rising Edge: It observes in synchronization with rising edges.
    • Falling Edge: It observes in synchronization with falling edges.

    While Rising Edge or Falling Edge mode, the synchronization standard value is defined by "Trigger". The time scale can be changed by "Skip".
